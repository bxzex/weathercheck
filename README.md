# Weather App

A beautiful, high-performance weather application built with React, TypeScript, and Tailwind CSS. This project provides real-time local forecasts and 5-day weather data using a clean, minimalist user interface.

## Features

- **Real-time Data:** Accurate current weather conditions.
- **5-Day Forecast:** Detailed daily temperature ranges and conditions.
- **Modern UI:** Minimalist aesthetic with a focus on typography and space.
- **Responsive:** Fully optimized for desktop and mobile devices.
- **Fast:** Built with Vite for near-instant development and optimized production builds.

## Tech Stack

- **Framework:** React 19
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Build Tool:** Vite
- **API:** WeatherAPI.com

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/bxzex/marketdata.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## Deployment

The project is configured for deployment to GitHub Pages. To deploy:

```bash
npm run deploy
```
